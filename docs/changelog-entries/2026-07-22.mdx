---
description: "Rule-Controlled Attachment Replies"
---

Ask Assistant to create an attachment-aware rule when you want incoming documents read before preparing a draft or reply. Ordinary rules leave attachments untouched, while enabled rules automatically switch to section-by-section processing for lengthy files.

- Enable attachment reading only on the draft and reply rules that need it.
- Use relevant facts from later sections of large PDFs and Word documents.
- Keep drafts cautious when a document is unreadable or only partly processed.
